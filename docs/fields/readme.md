# Fields

All field types available in the WordPress Settings Wrapper.

| Field | Class | Description |
|---|---|---|
| [InputText](/fields/input-text) | `InputText` | Text, color, email, password, date input |
| [InputNumber](/fields/input-number) | `InputNumber` | Numeric input with min/max/step |
| [InputPhone](/fields/input-phone) | `InputPhone` | Tel input with pattern/length validation |
| [InputUrl](/fields/input-url) | `InputUrl` | URL input |
| [Textarea](/fields/textarea) | `Textarea` | Multi-line text |
| [WYSIWYG](/fields/wysiwyg) | `WYSIWYG` | TinyMCE rich-text editor |
| [CheckBox](/fields/checkbox) | `CheckBox` | Single checkbox |
| [CheckBoxList](/fields/checkbox-list) | `CheckBoxList` | Multiple checkboxes |
| [Radio](/fields/radio) | `Radio` | Radio button group |
| [SelectBox](/fields/select-box) | `SelectBox` | Dropdown — single or multi-select |
| [UploadFile](/fields/upload-file) | `UploadFile` | File upload to uploads directory |
| [UploadMedia](/fields/upload-media) | `UploadMedia` | WordPress media library picker |
| [Custom](/fields/custom) | `Custom` | Fully custom HTML via callback |
| [AjaxButton](/fields/ajax-button) | `AjaxButton` | AJAX-triggered server action |
